> ⚠️ 请在使用本项目前完整阅读本文档。使用本项目即表示你已理解并同意以下声明。

## 1. 数据来源

本项目展示的所有数据来自:

- 用户上传的 Excel / CSV 文件
- 公开资料整理 (如社区维护的 955/965/996 名单)
- 用户投稿 (V2.0 上线后)

**本项目不采集以下数据:**

- 个人简历、手机号、邮箱、聊天记录
- 公司内部系统数据 (除非用户主动上传脱敏后的描述)
- 招聘网站的非公开信息

## 2. 数据真实性

- 本项目**不对数据真实性、完整性、时效性作任何保证**
- 数据可能存在错误、过时、片面或主观成分
- 同一条记录可能反映的是某员工某段时期的体验, 不代表公司当前的整体情况
- 证据字段多为描述性文本 (e.g. "员工手册截图"、"知乎帖子"), **不一定包含可访问 URL**

## 3. 不构成官方结论

本项目的所有展示 (包括工作强度等级、工作制度分类、可信度评级) **不代表**:

- 公司官方结论
- 政府监管部门认定
- 法律意义上的"违法"判定
- 学术研究的严谨结论

工作强度等级 (`risk_level`) 仅是基于上传数据文本的**关键词匹配**结果, 不应作为:
- 求职决策的唯一依据
- 法律诉讼的证据
- 媒体报道的引用来源
- 商业对比的权威数据

## 4. 同一公司差异巨大

同一家公司可能因为以下因素, 在不同记录中呈现出截然不同的作息:

- **城市差异**: 一线城市总部 vs 二三线城市分公司
- **部门差异**: 研发 vs 销售 vs 运营 vs 客服
- **岗位差异**: 应届生 vs 资深员工 vs 管理层
- **项目差异**: 平时 vs 项目冲刺期 vs 财报季
- **时间差异**: 2019 年 vs 2023 年 (公司制度可能已调整)
- **个人体验**: 不同员工对"加班"的感知阈值不同

本项目在 UI 与文档中**严格避免**以下绝对化表达:

| ❌ 错误表达                  | ✅ 正确表达                                |
| ---------------------------- | ------------------------------------------ |
| 某公司违法                   | 该记录显示存在高强度作息描述                |
| 某公司是黑心企业             | 数据来源记录称: 存在 996 / 大小周 / 单休情况 |

| 某公司不如某公司             | 本项目不做公司间横向排名                    |

## 6. 以官方信息为准

在做出任何决策前, 请以以下信息为准:

- **劳动合同** (明确工作时间、加班费计算)
- **公司正式发布的员工手册**
- **公司官方招聘 JD 中的工时说明**
- **面试时 HR / 用人部门的明确答复**
- **实际工作后的体验** (试用期 1-3 个月即可验证)
